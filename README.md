# PyXcel Workspace

This workspace contains the PyXcel desktop spreadsheet automation project.

Main runnable application:
- `pyxcel_complete/techxcel-har version`

Quick start:
1. Open terminal in `pyxcel_complete/techxcel-har version`
2. Install dependencies:
	`pip install -r requirements.txt`
3. Run:
	`python main.py`

Detailed documentation:
- See `pyxcel_complete/techxcel-har version/README.md`
